About Mods :

This Server uses the Fabric Mod Loader and all mods on this server REQUIRE fabric to be installed!

Guide: How to install Fabric loader

All mods on this server are strictly for Minecraft version 1.19 and no other Minecraft versions can connect :(

Drag the mods into the "mods" folder

Windows: C:\Users\"User"\AppData\Roaming\.minecraft\mods

Linux (Flatpak): /home/"User"/.var/app/com.mojang.Minecraft/.minecraft/mods

MacOS: ~/Library/Application Support/minecraft/mods

I strongly recommend using Prism Launcher for easily managing mods from Modrinth and CurseForge
